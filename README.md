# peptone-visuals

**Conformation Field** — a line-halftone synthesizer. Drop or paste an image and it becomes a moving field of parallel line paths, bent by a wave and modulated by the image.

**Live:** https://bas-taplow.github.io/peptone-visuals/

## Features
- Drag / paste / upload any image (opens on a generated noise field)
- Flow-field lines that follow image contours
- Duotone, thickness & warp modulation, analog wobble
- Bloom + film-grain finish
- Background: black / white / transparent
- Export: high-res PNG, still SVG, animated SVG (SMIL loop), 5s WebM loop
- Shareable settings codes + saved presets

Single self-contained `index.html` (p5.js loaded from CDN).
